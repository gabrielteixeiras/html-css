Lista feita com a TAG <ol> e dentro desta TAG utilizasse as TAGS em sequência <li> para ser formada a lista. Dentro da TAG é possível escolher o tipo de lista, entre eles temos 'A' para ordenar com letras maiúsculas, temos 'a' para ordenar com letras minúsculas, temos '1' para ordenação numérica, temos 'I' para ordenação em números romanos em maiúsculo, temos 'i' para ordenação em números romanos em minúsculo..
