Principais Formatações

Negrito / Destaque

Agora nessa frase, tenho um termo em destaque usando a tag <strong> </strong> (Com Semântica).

Itálico / Ênfase

Agora nessa frase, tenho um termo com Ênfase usando a tag <em> </em> (Com Semântica).

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark> </mark> (Com Semântica).

Texto Pequeno apenas, pois textos grandes é editável apenas no CSS3

Estamos criando texto pequeno nesse parágrafo, com a tag <small> </small> (Com Semântica).

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Com a tag <del> (Com Semântica).

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Com a tag <ins> (Com Semântica).

Texto Sobrescrito

Para inserir coisas do tipo: x20+3. Com a tag <sup> (Com Semântica).

Texto Subscrito

Para inserir coisas do tipo: H2O. Com a tag <sub> (Com Semântica).
